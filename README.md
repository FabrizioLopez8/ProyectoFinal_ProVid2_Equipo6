Descripción

BOLSAMON es un juego multijugador local para 3 jugadores inspirado en Pokémon.

Cada jugador controla una criatura elemental de tipo:

🔥 Fuego
💧 Agua
🌿 Planta

Los jugadores se enfrentan en una arena por turnos, seleccionando estratégicamente a qué rival atacar mientras administran una cantidad limitada de escudos defensivos.

El último jugador en permanecer con vida gana la partida.

Características
Combate por turnos.
3 jugadores simultáneos.
Ventajas y desventajas elementales.
Sistema de escudos limitados.
Selección estratégica de objetivos.
Interfaz visual con indicadores de vida, escudos y turnos.
